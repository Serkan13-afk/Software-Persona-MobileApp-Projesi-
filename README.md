# Mini Katalog Uygulaması

## 📖 Kısa Açıklama
Bu proje, Flutter temel bileşenlerini (widget yapısı, sayfa geçişleri, basit UI tasarımı, dinamik veri okuma ve modelleme) öğrenmek ve uygulamak amacıyla geliştirilmiş kapsamlı bir e-ticaret katalog simülasyonudur. Proje kapsamında; özel klasörleme mimarisi (models, views, services), ürünlerin internetten dinamik olarak listelendiği ana sayfa, argümanlarla veri taşıyan ürün detay ekranı ve temel bir sepete ekleme simülasyonu bulunmaktadır.

## 🛠 Kullanılan Teknolojiler ve Sürüm Bilgileri
* **Flutter Sürümü: 3.41.1
* **Dart Sürümü:  3.11.0 
* **Geliştirme Ortamı:** Visual Studio Code / Android Studio
* **Paketler:** `material.dart` (varsayılan), `http` (Ağ üzerinden API verisi çekmek için) ve `animate_do` (Widget geçiş animasyonları için) paketleri entegre edilmiştir.
